# BASTION Interface

Cinematic tactical sci-fi interface for the BASTION ecosystem.

## Structure

- `index.html` — стартова access-сторінка
- `styles/` — окремі CSS-модулі
- `scripts/` — JS-модулі
- `assets/` — фон, логотип, іконки, шрифти
- `pages/` — окремі сторінки входу, реєстрації, розробника
